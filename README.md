# Outros

Apps de terceiros / satélites do ecossistema CentralEXAuto (Geely IHU629G), servidos por OTA
ao AAExInstall / CentralEXAuto. Os 4 apps principais (CentralEXAuto, AAExInstall, AAExCarro,
CentralPhoneApp) e a infra (ConnAdaptor, CarplayMonitor, avisos) ficam em `swimapps/CentralEXAuto`.

Cada app tem um `<app>.json` (versão + apkUrl + sha256/md5) e o binário correspondente.
